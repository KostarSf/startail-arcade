# Система повышения уровня

При повышении уровня у игрока увеличиваются показатели:

- максимального здоровья
- максимальной энергии
- скорость восстановления энергии (прирост меньше чем общее количество энергии)
- повышается урон

# Ивенты

## Охотники за головами

Иногда на карте появляется отряд пиратов.
Пираты охотятся за игроком с наиболее высоким уровнем.
Пираты появляются недалеко от игрока (вылетают из ближайшей границы уровня)
Пираты по силе примерно равны этому игроку

Всем игрокам транслируется предупреждение - !! ПОВЫШЕННАЯ АКТИВНОСТЬ ПИРАТОВ В СЕКТОРЕ !!

Пираты спавнятся в отряде из 2х-4х кораблей. Движутся упорядоченной формацией в сторону игрока.
Атакуют других игроков оказавшихся рядом, пока либо их не убьют, либо игрока. После восстанавливаются на курс до игрока.
Если пираты слишком отстают от цели, которая не является для них основной (игрок с первого места), при атаке, они успокаиваются.
Пираты могут напасть друг на друга, если они несколько раз попадут друг по другу.
Если у пирата остается меньше 5% здоровья, пират отключается - корабль никуда не летит, движется по инерции, испускает искры дыма.

При уничножении пирата падает достаточное количество опыта.
Если отряд пиратов полностью уничножен, а игрок с первого места все еще жив,
следующая волна пиратов будет мощнее по силе и больше по количеству кораблей,
подкрепление придет быстрее обычного.
Если игрок с первого места умирает, накопленная мощность и таймер пиратов сбрасывается.

## Пираты
Как головорезы, только слабее и просто спавнятся в мире за пределами видимости игрока.
Могут сторожить собо ценные скопления ресурсов группой по несколько кораблей.

# Режим доп информации

При нажатии на Tab появляется:
- Радар вокруг игрока. Показывает противников и ценные предметы
- текущее/общее количество здоровья
- текущее/общее количество энергии, скорость восстановления
- базовый урон
- максимальную скорость
- другие баффы, если они будут
